# Анализ сервиса Яндекс Книги: расчет продуктовых метрик (SQL) и статистическая проверка гипотез (A/B тесты) на Python.

## 🛠 Технологический стек
*   **SQL:** расчет бизнес-метрик.
*   **Python (Pandas, Scipy, Statsmodels):** Разведочный анализ (EDA), статистический анализ выборок.
*   **Статистические методы и A/B-тестирование:** 
    *   **Тест Стьюдента (t-test):** Сравнение средних значений в группах.
    *   **Тест Уэлча:** Применение t-критерия при неравенстве дисперсий в выборках.
    *   **Z-тест пропорций:** Анализ конверсии и долевых показателей.
    *   **Интерпретация p-value:** Принятие решений на основе заданного уровня значимости ($\alpha = 0.05$).



## 📂 Структура репозитория
*   `Разработка AB-тестирования и анализ результатов.ipynb` — Jupyter Notebook с проверкой гипотезы и графиками.
*   `расчет бизнес-метрик(SQL)` — SQL-запросы.
