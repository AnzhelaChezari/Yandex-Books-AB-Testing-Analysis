# Комплексный анализ сервиса Яндекс Книги: расчет продуктовых метрик (SQL) и статистическая проверка гипотез (A/B тесты) на Python.

## 🛠 Технологический стек
*   **SQL:** Предобработка данных, расчет бизнес-метрик активности (чтение и прослушивание).
*   **Python (Pandas, Scipy):** Статистический анализ, проверка распределений, проведение t-теста.
*   **A/B-тестирование:** Анализ результатов экспериментов и интерпретация p-value.

## 📂 Структура репозитория
*   `Разработка AB-тестирования и анализ результатов.ipynb` — Jupyter Notebook с проверкой гипотезы и графиками.
*   `расчет бизнес-метрик(SQL)` — SQL-запросы.
